# 需求文档 — Prompt Recorder

## 项目概述

一个 Windows 桌面应用，用于记录和管理用户输入的提示词（prompts）。对接 DeepSeek API，自动为提示词分类并提供优化建议。

## 功能需求

### F1 — 提示词输入
- 用户可在文本框内自由输入/粘贴提示词
- 支持多行文本
- 提供「分析」按钮触发处理

### F2 — 智能分类
- 点击分析后，调用 DeepSeek API 对提示词进行分类
- 优先匹配用户在设置中预设的自定义分类
- 若无匹配的分类，AI 可自由发挥推荐分类
- 分类结果以标签形式展示

### F3 — 优化建议（可开关）
- 分析时间调用 DeepSeek API 对原始提示词进行通用优化
- 优化方向：语法、表达清晰度、结构、完整性
- 返回优化后的文本及优化说明
- 用户可在设置中开启/关闭此功能
- 关闭后仅分类不优化，直接保存原文

### F4 — 结果展示
- 中间面板展示：原始文本 vs 优化文本（并排对比）
- 显示分类标签
- 显示优化说明文字

### F5 — 确认机制
- 用户可选择「采用优化版」或「保留原文」
- 选择后记录存入数据库

### F6 — 历史记录
- 右侧面板展示已保存的提示词列表
- 支持关键词搜索过滤
- 展示：分类标签、原文摘要、保存时间

### F7 — 设置管理
- API Key 配置（密码输入框）
- 模型选择
- 优化建议开关
- 自定义分类管理（增删）

## 非功能需求

- 所有数据本地存储（SQLite）
- UI 风格参考 Claude：简洁、暖色调、低视觉噪音
- 代码分层清晰，服务层与 UI 层解耦
- 配置文件驱动，便于扩展
